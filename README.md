# Kramerius Music

Jednotný webový portál pro zvukové dokumenty systému [Kramerius](https://github.com/ceskaexpedice/kramerius)

Portál je dostupný na http://music.kramerius.org a obsahuje zvukové dokumenty z Krameriů Národního muzea, Moravské zemské knihovny v Brně, Krajské knihovny Františka Bartoše ve Zlíně a Městské knihovny v Praze.

Kramerius Music umožňuje výhledávat a procházet alba, umělce, žánry a sklady. Přehrávat skladby přes integrovaný přehrávač, vytvářet frontu skladeb a po přihlášení si mohou uživatelé vytvářet vlastní playlisty a přidávat alba a umělce do své knihovny.

Součástí portálu je i [serverová vrstva](https://github.com/ceskaexpedice/kramerius-music-backend), která sjednocuje data z jednotlivých Krameriů a ukládá uživatelské informace - playlisty a knihovnu.
